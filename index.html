EduManage — Course Management System

Backend Full Stack Deployed
Live Demo GitHub

EduManage is a full-stack web application designed to digitize and streamline academic operations for schools and colleges. It replaces manual paper-based processes for course enrollment, attendance tracking, grade recording and notice distribution with a centralized digital platform accessible from any device.

Problem It Solves

Educational institutions traditionally manage courses, attendance and grades using paper registers, Excel sheets and manual processes. This leads to data loss, delays, difficulty in tracking student performance and poor communication between teachers and students. EduManage solves this by providing a single platform where admins can manage users, teachers can track attendance and grade assignments, and students can monitor their own progress — all in real time.

Secure login interface with JWT authentication and password visibility toggle

Key Features

  • Role-based access — Admin, Teacher and Student dashboards
  • JWT-secured login with password visibility toggle
  • Course creation, browsing and one-click enrollment
  • Real-time attendance marking with Present/Absent/Late/Excused status
  • Assignment creation and grade management with letter grade calculation
  • Notice board with pinned announcements and category badges
  • Dark and Light theme support
  • Fully deployed and accessible online
